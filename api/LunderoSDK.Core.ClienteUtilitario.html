public static Task<RespuestaApi<T>> RealizarSolicitud<T>(HttpMethod metodo, string url, ClienteOpciones opciones, object contenidoDatos = null, ILoggerSdk logger = null, bool esRespuestaEnvuelta = true, CancellationToken tokenCancelacion = default)

Parameters

metodo HttpMethod

El método HTTP a utilizar (GET, POST, etc.).

url string

La URL a la que se realizará la solicitud.

opciones ClienteOpciones

Las opciones del cliente, como encabezados y parámetros de consulta.

contenidoDatos object

El contenido de los datos a enviar en la solicitud (opcional).

logger ILoggerSdk

Instancia del logger a utilizar (opcional).

esRespuestaEnvuelta bool

Indica si la respuesta está envuelta en un objeto de respuesta API.

tokenCancelacion CancellationToken

El token de cancelación para la solicitud (opcional).

Returns

Task<RespuestaApi<T>>

Una tarea que representa la operación asincrónica. El resultado contiene la respuesta deserializada.

Type Parameters

T

El tipo de la respuesta esperada.
